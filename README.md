# Attendance Management System

## 📌 Project Overview

Attendance Management System is a web-based application developed using PHP and MySQL. It helps manage student records, track attendance, generate reports, and calculate attendance percentages efficiently.

## 🚀 Features

* Admin Login Authentication
* Add Student
* View Student Details
* Edit Student Information
* Delete Student Records
* Mark Daily Attendance
* Attendance Report Generation
* Attendance Percentage Calculation
* Dashboard Statistics
* Secure Session Management
* Logout Functionality

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML
* CSS
* XAMPP
* Git & GitHub

## 🗄️ Database

Database Name:

attendance_system

Tables:

* students
* attendance
* users

## 📊 Modules

### Student Management

* Add Student
* View Student
* Edit Student
* Delete Student

### Attendance Management

* Mark Attendance
* Attendance Report
* Attendance Percentage

### Authentication

* Login
* Logout
* Session Security

## 🎯 Project Objective

To automate student attendance management, reduce manual work, maintain attendance records, and generate reports efficiently using a web-based system.
